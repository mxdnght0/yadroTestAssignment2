# namecount

Консольная утилита на Go, которая принимает путь к файлу с именами и выводит количество вхождений каждого имени. Реализована трёхслойная архитектура: `repository` читает файл построчно через `bufio.Scanner` без загрузки в память целиком, `service` считает частоты и сортирует результат, `handler` форматирует вывод. По умолчанию имена выводятся в порядке возрастания частоты, флаг `-desc` меняет порядок на убывающий. Каждый слой покрыт юнит-тестами с моками соседних слоёв. CI на GitHub Actions запускает сборку и тесты.
